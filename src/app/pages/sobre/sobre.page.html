Sobre

HelpDesk Pro

Versão 1.0.0

Disciplina Desenvolvimento de Aplicativos Mobile
Objetivo do Projeto Desenvolver um sistema de chamados técnicos aplicando conceitos de páginas, navegação, formulários, arrays, services e componentes visuais do Ionic.
Tecnologias Utilizadas Ionic Framework, Angular, TypeScript, SCSS, Capacitor

Recursos Implementados

  • ✅ Tela de Login com validação
  • ✅ Menu Principal com navegação
  • ✅ Cadastro de Chamados com formulários ngModel
  • ✅ Listagem com filtros e busca
  • ✅ Detalhes e atualização de status
  • ✅ Cadastro e listagem de técnicos
  • ✅ Resumo com estatísticas por status e prioridade
  • ✅ Dados armazenados em arrays (sem banco)
  • ✅ Compartilhamento via Services Angular
  • ✅ Layout responsivo para mobile

Feito com usando Ionic + Angular
